Description

In this crash course, we explore the Ruby programming language and use it to answer some burning questions about America's most popular superhero team - The Avengers.

  • How likely are you to die when you join The Avengers?
  • If you die, how likely are you to come back?
  • Do male Avengers die more often than women?
  • Wait. Just how many folks have been in the Avengers anyway?
  • And more!

Along the way, you'll learn about:

  • Variables
  • Arrays
  • Hashes
  • Data Structures
  • CSVs
  • Enumerable
  • Comic books

Of note: We won't be delving into Ruby on Rails, the popular Ruby-based web application framework. Ruby itself is fun enough to fill an evening.
